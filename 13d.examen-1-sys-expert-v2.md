### Question 4 — 4 points

Écrivez une fonction appelée `conditions_sont_verifiees`.

Cette fonction doit recevoir une règle et la base de faits.

Elle doit retourner `True` si toutes les conditions de la règle sont présentes dans les faits avec les bonnes valeurs.

Elle doit retourner `False` sinon.

Exemple attendu :

```python
def conditions_sont_verifiees(rule, facts):
    for condition, valeur_attendue in rule["conditions"].items():
        if facts.get(condition) != valeur_attendue:
            return False
    return True
```
